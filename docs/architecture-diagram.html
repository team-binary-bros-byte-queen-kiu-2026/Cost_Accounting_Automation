# Step Detail
1Upload imageJPEG / PNG / WebP, max 10 MB via drag-and-drop on the frontend
2POST /analyzeMultipart form to FastAPI. Rate limiter checks IP (20/min)
3VisionAgentImage base64-encoded, sent to gemini-2.5-flash via OpenRouter. Returns JSON with building type, construction stage, and component list with quantities
4EstimationAgentEach component looked up in SQLite by name or category. Unit price × quantity = subtotal. Unpriced items shown as 0 with "price not found" note
5Labor + total35% of materials added as labor estimate. grand_total_gel computed
6Approval gateIf total > 500,000 GEL → HumanReview node appends warning, sets approval_required = true
7SessionStoreEstimate stored in-memory under UUID session_id. Returned to frontend alongside model metadata
8Report renderedFrontend displays itemized table. session_id stored for follow-up chat
9POST /chat/streamUser question + session_id sent. Rate limit: 60/min
10ChatAgent builds contextLoads estimate from SessionStore. Retrieves top-3 RAG chunks from ChromaDB. Injects both into system prompt
11Streaming responseclaude-3-5-haiku streams tokens as SSE data: {"token":"…"} events. Full reply saved to session history. Ends with data: [DONE]
